# Laneway Employee Portal — Frontend

React (Vite) frontend for the Laneway Employee Management Portal.

Backend repository: https://github.com/ashwany01/laneway-employee-portal

## Setup
1. `npm install`
2. `npm run dev`
3. Runs on `http://localhost:5173`
4. Requires the backend running on `http://localhost:8080` (see backend repo for setup)

## Pages
- Login
- Employee Directory (search, filter by location/status, pagination)
- Employee Profile (view/edit)
- Organization Hierarchy (management chain + direct reports)
- Bench/Utilization Dashboard

## Notes
- Visual design is intentionally simple (inline styles), per assessment priorities — backend correctness was prioritized over frontend polish.
